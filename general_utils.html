get_free_var_names

 get_free_var_names (term_list:Sequence, type_list:Sequence)

@raise Exception: if length of term list doesn’t match the length of type list.

Type Details
term_list typing.Sequence a list of terms
type_list typing.Sequence a list of the term types
Returns typing.Set[str] a set of all the free variable names in term_list
Example 
term_list = ["X", 10, "Z"]
type_list = [DataTypes.free_var_name, DataTypes.integer, DataTypes.free_var_name]    
print(get_free_var_names(term_list, type_list))
{'Z', 'X'}

position_freevar_pairs

 position_freevar_pairs (relation:Union[spannerlib.ast_node_types.Relation
                         ,spannerlib.ast_node_types.IERelation])
Type Details
relation typing.Union[spannerlib.ast_node_types.Relation, spannerlib.ast_node_types.IERelation] a relation (either a normal relation or an ie relation)
Returns typing.List[typing.Tuple[int, str]] a list of all (index, free_var) pairs based on term_list
Example

relation = Relation("relation1",term_list, type_list)

print(position_freevar_pairs(relation))
[(0, 'X'), (2, 'Y'), (4, 'Z')]

get_input_free_var_names

 get_input_free_var_names (relation:Union[spannerlib.ast_node_types.Relati
                           on,spannerlib.ast_node_types.IERelation])
Type Details
relation typing.Union[spannerlib.ast_node_types.Relation, spannerlib.ast_node_types.IERelation] a relation (either a normal relation or an ie relation)
Returns typing.Set[typing.Any] a set of the free variables used as input terms in the relation.
Note

if the input is relation it returns empty set as the regular relation don’t have input free variables, if the input is ie-relation it returns it’s input free variables

Example 
input_term_list = ["X", "Y"]
input_type_list = [DataTypes.free_var_name, DataTypes.integer]

ie_relation = IERelation("relation1",input_term_list, input_type_list, [], [])

print(get_input_free_var_names(ie_relation))
{'X'}

get_free_var_to_relations_dict

 get_free_var_to_relations_dict (relations:Set[Union[spannerlib.ast_node_t
                                 ypes.Relation,spannerlib.ast_node_types.I
                                 ERelation]])

Finds for each free var in any of the relations, all the relations that contain it. also return the free vars’ index in each relation (as pairs).
for example:
relations = [a(X,Y), b(Y)] -> dict = {X:[(a(X,Y),0)], Y:[(a(X,Y),1),(b(Y),0)]}

Type Details
relations typing.Set[typing.Union[spannerlib.ast_node_types.Relation, spannerlib.ast_node_types.IERelation]] a set of relations
Returns typing.Dict[str, typing.List[typing.Tuple[typing.Union[spannerlib.ast_node_types.Relation, spannerlib.ast_node_types.IERelation], int]]] a mapping between each free var to the relations and corresponding columns in which it appears
Example 
term_list_1 = ["X", "abc", "Y", "def", "Z"]
type_list_1 = [DataTypes.free_var_name, DataTypes.string, DataTypes.free_var_name, DataTypes.integer, DataTypes.free_var_name]

relation_1 = Relation("relation_1",term_list_1, type_list_1)

term_list_2 = ["X", "Y"]
type_list_2 = [DataTypes.free_var_name, DataTypes.free_var_name]

relation_2 = Relation("relation_2",term_list_2, type_list_2)
relations_set = {relation_1,relation_2}
get_free_var_to_relations_dict(relations_set)
{'Y': [(relation_1(X, "abc", Y, def, Z), 2), (relation_2(X, Y), 1)],
 'Z': [(relation_1(X, "abc", Y, def, Z), 4)],
 'X': [(relation_1(X, "abc", Y, def, Z), 0), (relation_2(X, Y), 0)]}

type_check_rule_free_vars

 type_check_rule_free_vars (rule:spannerlib.ast_node_types.Rule,
                            symbol_table:spannerlib.symbol_table.SymbolTab
                            leBase)

Free variables in rules get their type from the relations in the rule body.
it is possible for a free variable to be expected to be more than one type (meaning it has conflicting types).
for each free variable in the rule body relations, this method will check for its type and will check if it has conflicting types

Type Details
rule Rule The rule to be checked
symbol_table SymbolTableBase a symbol table (used to get the schema of the relation)
Returns typing.Tuple[typing.Dict[str, spannerlib.primitive_types.DataTypes], typing.Set[str]] a tuple (free_var_to_type, conflicted_free_vars) where
free_var_to_type: a mapping from a free variable to its type
conflicted_free_vars: a set of all the conflicted free variables
Note

this function updates free_var_to_type’s mapping if it finds new free variables in term_list
